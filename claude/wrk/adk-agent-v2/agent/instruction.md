# Summarize Input Files

## Your Task

List all files in the input folder, read each one, and write a structured
summary to `output/summary.md`.

## Steps

1. Call `list_input_files` to discover what files are available.
2. Call `read_input_file` for each file returned.
3. Write a Markdown summary to `output/summary.md` using `write_output`.
   Include: file name, approximate size, and a 2-3 sentence description of
   the contents.
