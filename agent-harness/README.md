These agent harnesses were developed via Claude Code and/or Cursor Agent.
Anthropic Sonnet or Composer are used as the AI models.

- claude-agent-sdk, openai-agents, google-adk are all open source (MIT or Apache 2.0).
- Only claude-agent-sdk bundles the claude CLI binaries, and is proprietary.  
- The others just call cloud APIs over HTTPS, no binaries involved.
