CHART EXAMPLES GENERATED

The scripts create the following types of visualizations:

ASCIICHARTPY (Line/Area Charts):
  Format: Text-based line drawing with ASCII characters
  Examples:
    ├─ Survival by gender (trends)
    ├─ Age distribution
    ├─ Family size impact
    └─ Port comparison

PLOTEXT (Bar Charts & Histograms):
  Format: Unicode block characters (▇▇▇)
  Examples:
    ├─ Survival rate by class (horizontal bars)
    ├─ Fare distribution (histogram)
    ├─ Port survival rate
    └─ Class & gender survival

TPLOT (Simple Bar Charts):
  Format: Unicode blocks with Unicode rendering
  Examples:
    ├─ Simple gender survival
    ├─ Class survival rates
    ├─ Age groups
    ├─ Fare ranges
    ├─ Port distribution
    ├─ Family size
    └─ Cabin deck distribution

================================================================================
LIBRARY COMPARISON
================================================================================

ASCIICHARTPY
  Pros:  Good for trends, lightweight, smooth curves
  Cons:  Limited chart types, smaller output
  Best:  Time series and line trends
  Example:
    468.00  ┤
    428.11  ┤╭
    388.22  ┤│
    ...

PLOTEXT
  Pros:  Professional output, multiple charts, colors, customizable
  Cons:  More setup, medium complexity
  Best:  Dashboards, reports, multiple visualizations
  Example:
    First  ▇▇▇▇▇▇▇▇▇▇▇▇▇▇ 62.96
    Second ▇▇▇▇▇▇▇▇▇▇ 47.28
    Third  ▇▇▇▇▇ 24.24

TPLOT
  Pros:  Very simple, lightweight, fast
  Cons:  Limited to bars, no colors, minimal features
  Best:  Quick visualizations, minimal setup
  Example:
    First Class  ███████████████████████████████ 63.0%
    Second Class ███████████████████ 47.3%
    Third Class  ███████ 24.2%

================================================================================
RECOMMENDATIONS
================================================================================

Use ASCIICHARTPY when:
  • Displaying trends over time
  • Need line charts in terminal
  • Want smooth curve rendering
  • Minimal dependencies needed

Use PLOTEXT when:
  • Building terminal dashboards
  • Need professional-looking output
  • Want multiple chart types
  • Need color support
  • Creating reports

Use TPLOT when:
  • Need quick visualization
  • Minimal dependencies desired
  • Simple bar charts sufficient
  • Maximum performance needed
  • Just exploring data

For production dashboards: PLOTEXT (most feature-complete)
For data exploration:      TPLOT or ASCIICHARTPY
For time series:           ASCIICHARTPY
For reports:               PLOTEXT
