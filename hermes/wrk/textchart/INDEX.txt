WHAT IS THIS PROJECT?
================================================================================

This project demonstrates how to create text-based charts in Python using three
popular terminal plotting libraries:

• ASCIICHARTPY  - ASCII line charts
• PLOTEXT       - Professional bar charts & histograms  
• TPLOT         - Simple Unicode bar visualizations

The project includes:
✓ Real-world dataset (Titanic survival data - 891 passengers)
✓ Complete statistical analysis
✓ 9+ different chart examples
✓ Full documentation
✓ Working Python scripts ready to run

================================================================================
LIBRARIES EXPLAINED
================================================================================

📈 ASCIICHARTPY
   Use for: Line charts, trends, time series
   Style: ASCII line drawing
   Example:
     468.00  ┤
     428.11  ┤╭
     388.22  ┤│
   Complexity: Low
   Pro: Lightweight, smooth curves
   Con: Limited chart types

📊 PLOTEXT
   Use for: Dashboards, reports, professional output
   Style: Unicode block bars (▇▇▇)
   Example:
     First  ▇▇▇▇▇▇▇▇▇▇▇▇▇ 62.96
     Second ▇▇▇▇▇▇▇▇▇▇ 47.28
     Third  ▇▇▇▇▇ 24.24
   Complexity: Medium
   Pro: Best-looking, many chart types, colors
   Con: More setup

📉 TPLOT
   Use for: Quick visualization, minimal setup
   Style: Unicode block bars (█)
   Example:
     First Class  ███████████████████████████ 63.0%
     Second Class ███████████████ 47.3%
     Third Class  ████████ 24.2%
   Complexity: Very Low
   Pro: Simplest, fastest
   Con: Limited features

================================================================================
WHEN TO USE EACH LIBRARY
================================================================================

ASCIICHARTPY:
  ✓ When showing trends over time
  ✓ When you need line charts
  ✓ When minimal dependencies matter
  ✗ Don't use for simple bar comparisons

PLOTEXT:
  ✓ When building dashboards
  ✓ When presentation quality matters
  ✓ When you need multiple chart types
  ✗ Don't use if minimal dependencies needed

TPLOT:
  ✓ When you need quick results
  ✓ When dependencies should be minimal
  ✓ When simple bars are enough
  ✗ Don't use for complex visualizations

================================================================================
CUSTOMIZATION TIPS
================================================================================

Want to modify the scripts?

Edit chart parameters:
  Line 70 in analyze_titanic.py:
  print(asciichartpy.plot(values, {'height': 10, 'width': 50}))
  
  Change height: 10 → 20 (taller chart)
  Change width: 50 → 100 (wider chart)

Add new analysis:
  Add this after existing analysis:
  avg_age_by_class = df.groupby('Pclass')['Age'].mean()
  print(avg_age_by_class)

Filter different data:
  df_young = df[df['Age'] < 18]
  young_survival = df_young['Survived'].mean()

Create new CSV:
  df.groupby('Sex')['Survived'].sum().to_csv('gender_summary.csv')

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "ModuleNotFoundError: No module named 'pandas'"
Solution: Activate environment: source venv/bin/activate

Problem: Charts are misaligned
Solution: Make terminal wider (at least 80 chars)

Problem: Can't download data
Solution: Run again (temporary network issue), or use existing CSV

Problem: "python3: command not found"
Solution: Install Python 3.8+, or use "python" instead

Problem: Venv doesn't work
Solution: Run: python3 -m venv venv
         Then: source venv/bin/activate
