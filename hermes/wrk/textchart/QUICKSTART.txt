PROJECT LOCATION: /root/hermes/wrk/textchart/

================================================================================
FIRST TIME SETUP (One-time only)
================================================================================

1. Navigate to project directory:
   $ cd /root/hermes/wrk/textchart

2. Create Python virtual environment:
   $ python3 -m venv venv

3. Activate environment:
   $ source venv/bin/activate

4. Install dependencies:
   $ pip install pandas requests asciichartpy plotext tplot

Done! You're ready to run the scripts.

================================================================================
RUNNING THE SCRIPTS
================================================================================

BEFORE EVERY SESSION - Activate the environment:
   $ cd /root/hermes/wrk/textchart
   $ source venv/bin/activate

THEN CHOOSE A SCRIPT:

Option 1: MAIN ANALYSIS (Recommended for first run)
   $ python3 analyze_titanic.py
   
   This will:
   • Download Titanic dataset (891 passengers)
   • Show statistical analysis
   • Generate 9 charts in terminal
   • Create titanic.csv for future use
   Duration: ~5 seconds

Option 2: TPLOT EXAMPLES
   $ python3 tplot_examples.py
   
   This will:
   • Use existing titanic.csv
   • Create 8 simple bar charts
   • Show data distributions
   Duration: ~2 seconds

Option 3: LIBRARY COMPARISON
   $ python3 comparison.py
   
   This will:
   • Show same data in 3 different libraries
   • Display visual differences
   • Include feature comparison table
   Duration: ~3 seconds

================================================================================
LIBRARY SUMMARY
================================================================================

ASCIICHARTPY - Line Charts
  Best for: Trends, time series
  Output: ASCII line drawings
  Complexity: Low
  
PLOTEXT - Bar Charts & Histograms
  Best for: Dashboards, reports
  Output: Unicode bars (▇▇▇)
  Complexity: Medium
  Features: Colors, multiple chart types
  
TPLOT - Simple Bars
  Best for: Quick visualization
  Output: Unicode blocks (█)
  Complexity: Very Low
  Features: Minimal setup needed

================================================================================
TIPS & TRICKS
================================================================================

1. Maximize Terminal Width
   Charts adapt to terminal size. Make your terminal wider for better display.

2. Run All Scripts in Sequence
   $ python3 analyze_titanic.py && python3 tplot_examples.py && python3 comparison.py
   
   This shows all three libraries in action.

3. Customize the Data
   Edit the scripts to:
   • Change age bins
   • Filter by different criteria
   • Add new calculations
   • Create different chart types

4. Use Different CSV Files
   Modify line in tplot_examples.py:
   $ df = pd.read_csv('your_file.csv')  # Replace titanic.csv

5. Scroll Back in Terminal
   Some output may scroll off screen. Scroll up to see all charts.

================================================================================
TROUBLESHOOTING
================================================================================

Issue: "ModuleNotFoundError: No module named 'pandas'"
Solution: Make sure venv is activated: source venv/bin/activate

Issue: "Command not found: python3"
Solution: Make sure Python 3.8+ is installed: python3 --version

Issue: Charts look ugly/misaligned
Solution: 
  • Make terminal wider (at least 80 chars)
  • Check terminal encoding is UTF-8
  • Try maximizing terminal window

Issue: Network error downloading data
Solution:
  • Check internet connection
  • Try running again (temporary network issue)
  • titanic.csv is created after first run, subsequent runs use local copy
